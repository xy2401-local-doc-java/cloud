저장소 관리자

새 사용자 롤

모든 사용자

두 개의 새 사용자 롤(저장소 관리자 및 개발자)이 추가되었고 롤 설명서가 향상되었습니다. 롤 in Oracle Content and Experience Cloud 관리를 참조하십시오.

메뉴 구조의 변경사항

모든 이전 사용자

서비스의 탐색 옵션이 간소화되었습니다. 표시되는 항목은 지정된 롤과 서비스가 표준 서비스 또는 엔터프라이즈 서비스인지에 따라 달라집니다.

  • 문서를 누르면 파일 관리 기능이 표시됩니다.

  • 대화를 누르면 대화 관련 기능에 액세스할 수 있습니다.

  • 사이트를 누르면 모든 사이트 구축 기능에 액세스할 수 있습니다.

  • 자산을 누르면 디지털 자산이 표시됩니다.

  • 개발자를 누르면 템플리트, 테마, 레이아웃에 액세스할 수 있습니다.

  • 분석을 누르면 서비스 관련 통계에 액세스할 수 있습니다.

관리 롤이 지정된 경우 추가 옵션에 설정(서비스 전체 설정에 사용) 및 자산(저장소, 콘텐츠 유형, 채널, 지역화 정책 설정에 사용)이 포함됩니다.

사람, 그룹, 모음, 콘텐츠 항목

모든 이전 사용자

이 기능은 이제 다른 메뉴 구조에 통합되었습니다.

  • 사람 개인을 보려면 대화에서 그 사람의 아바타를 누르면 됩니다.

  • 그룹 기능은 이제 사용자 메뉴를 통해 액세스할 수 있습니다.

  • 모음은 이제 개별 저장소에서 관리됩니다.

  • 콘텐츠 항목은 개별 저장소에서 관리됩니다. 콘텐츠 유형은 관리자가 [관리] 메뉴에서 [자산]을 눌러 생성할 수 있습니다.

새 JET 시작 템플리트

사이트 사용자

이제 사이트 생성에 사용할 수 있는 Oracle JET(JavaScript 확장 툴킷) 템플리트가 있습니다. JET는 오픈 소스 JavaScript 라이브러리 모음을 사용하는 모듈식 오픈 소스 툴킷입니다. 자세한 내용은 Oracle Jet를 참조하십시오.

사이트 향상

사이트 사용자
사이트를 생성하는 사용자를 위해 여러 기능이 향상되었습니다.

  • 사용자정의 구성요소 내에서 디지털 자산을 선택하고 사용할 수 있습니다.

  • 개발자는 사이트 구성요소에서 사용되는 편집기 도구모음을 사용자정의할 수 있습니다.

  • 이제 맵 구성요소의 옵션으로 Google Maps를 선택할 수 있습니다.

  • 콘텐츠 검색 구성요소가 향상되어 검색 결과가 빨라졌습니다.

  • 이제 Oracle JET를 페이지 레이아웃 및 페이지에서 사용할 수 있습니다.

모바일 용도의 사이트 향상

사이트 사용자
다음과 같이 모바일 설정용 사이트 생성을 위한 기능이 향상되었습니다.

  • 모바일 설정용 사이트를 미리 볼 때 선택한 모바일 장치의 크기에 반응하여 레이아웃과 화면이 최적화됩니다.

  • 페이지, 콘텐츠 항목 또는 콘텐츠 목록은 모바일 장치용으로 설계된 경우 대체 레이아웃을 사용할 수 있습니다.

  • 모바일 장치의 고유 맵에서 링크를 여는 맵 링크를 생성할 수 있습니다.

탐색 패널에 대한 변경사항

모바일 사용자

모바일 사용자용 탐색 패널이 변경되었습니다. 탐색 옵션 이름이 변경되었고 아이콘이 제거되었습니다. 이제 설정, 도움말 및 정보를 패널 아래쪽에서 액세스합니다. 몇 가지 탐색 변경사항은 다음과 같습니다.

  • 계정 사용: 계정 사이를 전환하려면 패널 아래쪽에 있는 계정 이름을 누릅니다.

  • 쪽지: 다른 사용자에게 쪽지를 전송하려면 해당 사용자의 아바타를 누르고 이 사용자의 프로파일 페이지에서 쪽지를 선택합니다.

  • 파일 다운로드: iOS 장치에서 파일을 다운로드하려면 파일을 확인하고 을 누릅니다. 오프라인 형식으로 저장을 선택합니다. 파일은 오프라인 영역에 저장되고 홈 페이지 머리글에 있는 메뉴를 통해 액세스됩니다.

자세한 내용은 장치에 대한 도움말 파일을 참조하십시오.

새 가입 모델

모든 사용자

Oracle UCM(범용 크레딧 모델)을 사용하는 Oracle Content and Experience Cloud의 첫번째 릴리스입니다. 2018년 3월, Oracle Content and Experience Cloud는 범용 크레딧 모델로 이동했습니다. 2018년 3월 이후에 가입을 구매한 경우 범용 크레딧 가입을 보유하며  Oracle Identity Cloud Service(IDCS)에 클라우드 계정이 있습니다. 2018년 3월 이전에 가입을 구매한 경우 종량제 또는 정액제 가입을 보유하며 기존 클라우드 계정이 있습니다.

인터페이스 또는 기능의 경우 사용자에 대한 최소한의 변경사항이 있습니다. 관리 작업 몇 가지가 변경되었습니다. 이러한 항목 및 기타 항목은 Oracle Content and Experience Cloud 관리를 참조하십시오.

  • 범용 크레딧 가입으로 Oracle Content and Experience Cloud 인스턴스 생성

  • 사용자 및 해당 롤 관리

  • 서비스 작업 모니터

  • 시간당 활성 사용자 이해

  • Oracle Cloud에 가입

기능 롤 설명
사이트: 콘텐츠 템플리트 사이트 사용자 및 관리자

이제 콘텐츠 템플리트를 사용하여 콘텐츠 모델을 공유할 수 있습니다. 여기에는 콘텐츠 레이아웃, 콘텐츠 항목 및 콘텐츠 모델(예: 샘플 콘텐츠)을 지원하는 데 필요한 디지털 자산이 포함됩니다.

콘텐츠 템플리트를 사용하면 엔터프라이즈 사용자가 템플리트에서 콘텐츠 유형 및 레이아웃을 사용하여 편집 콘텐츠 생성을 시작할 수 있습니다. 엔터프라이즈 사용자는 이 콘텐츠를 사이트 또는 다른 가용 채널에 게시할 수 있습니다.

기존 사이트에서 콘텐츠 템플리트를 생성합니다. 그런 다음 다른 사용자와 공유할 수 있습니다. 콘텐츠 템플리트는 게시된 콘텐츠 항목 및 디지털 자산을 사용하여 사이트에서 생성해야 합니다.

콘텐츠가 포함된 패키지에서 사이트 템플리트를 임포트하는 경우 콘텐츠 유형이 생성되지 않습니다. 콘텐츠 유형은 콘텐츠 템플리트를 기반으로 사이트가 생성되는 경우 생성됩니다. 따라서 콘텐츠 유형의 소유자는 사이트 템플리트를 임포트한 사용자가 아닙니다. 콘텐츠 유형의 소유자는 해당 템플리트를 기반으로 첫번째 사이트를 생성한 사용자입니다. 해당 사용자는 필요에 따라 다른 사용자와 템플리트를 공유할 수 있습니다.
사이트: 향상된 사이트 게시 성능 사이트 사용자

사이트 활성화는 시간이 오래 걸리는 프로세스일 수 있지만 새 기능은 동적 사이트 폴더에서 사이트 파일을 직접 제공합니다.
사이트: 콘텐츠 목록 구성요소에 대한 변경 사이트 사용자

콘텐츠 목록 구성요소는 추가 질의 필드가 제공되어 개선되었으며 사용자가 향상된 필터 정보를 사용할 수 있습니다. 예를 들어 검색 연산자(예: eq(같음), le(보다 작거나 같음))를 사용하여 콘텐츠 목록에서 항목을 필터링할 수 있습니다.
데스크톱 앱 업로드 성능 모든 사용자

안정적인 네트워크에서 콘텐츠 업로드 속도가 더 빨라졌습니다. 이전에는 제한이 있었지만 이제 5GB가 넘는 파일을 업로드할 수 있습니다.
