Изменения в структуре меню

Все бывшие пользователи

Для данного сервиса были упрощены параметры навигации. Отображаемые элементы зависят от назначенной вам роли и типа вашего сервиса: стандартный или корпоративный.

  • Если нажать Документы, отображаются функции управления файлами.

  • Для доступа к функциям, связанным с диалогами, следует нажать Диалоги.

  • Для доступа ко всем функциям, связанным с созданием сайтов, следует нажать Сайты.

  • Если нажать Активы, отображаются цифровые активы.

  • Для доступа к шаблонам, темам и макетам следует нажать Разработчик.

  • Для доступа к статистике, связанной с сервисом, следует нажать Аналитика.

Если вам назначена административная роль, появляются дополнительные возможности: Настройки, используется для настройки параметров в масштабе всего сервиса, и Активы, используется для настройки репозиториев, типов содержимого, каналов и политик локализации.

Новый начальный шаблон JET

Сайты: шаблоны содержимого Пользователи и администраторы сайтов

Шаблоны содержимого теперь можно использовать для обмена моделями содержимого. Модели содержимого включают в себя макеты содержимого, элементы содержимого и цифровые активы, необходимые для поддержки модели содержимого (например, образцы содержимого).

С помощью шаблонов содержимого корпоративные пользователи могут создавать некоммерческое содержимое с типами содержимого и макетами, заданными в шаблоне. Пользователи могут публиковать это содержимое на сайтах и других доступных каналах.

Шаблон содержимого создается на основе существующего сайта. Шаблоном можно поделиться с другими пользователями. Шаблоны содержимого следует создавать на основе сайтов с опубликованными элементами содержимого и цифровыми активами.

Обратите внимание, что при импорте шаблона сайта из пакета с содержимым не создаются типы содержимого. Они создаются, когда сайт создается на основе шаблона содержимого. Следовательно, типы содержимого не принадлежат пользователю, который импортирует шаблон сайта. Они принадлежат пользователю, который создает первый сайт на основе этого шаблона. При необходимости этот пользователь может поделиться шаблоном с другими.

Разработчики
