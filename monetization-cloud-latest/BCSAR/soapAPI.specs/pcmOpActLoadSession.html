pcmOpActLoadSession

Visibilitypublic
Transactionrequired

Description

Create, rate, and record a session event as a single operation. This operation is valuable as a tool to load sessions in real-time.

Input details

pcmOpActLoadSession request specifications. Used to record an entire user session. Inherited info for a specific session type is passed via a generic substruct. Specs for the inherited info for each session type are given elsewhere.

Since the PIN_FLD_EXTENDED_INFO substruct is reserved for special functionality such as IP Telephony.
