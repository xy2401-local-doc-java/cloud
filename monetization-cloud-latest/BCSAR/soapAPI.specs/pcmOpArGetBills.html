pcmOpArGetBills

Componentfm_ar
Visibilitypublic
Transactionrequired

Description

This operation is used to get the list of bills of a given Bill-info based on search criteria. This operation also accepts a Bill poid for the search. Notes on the alternation to the operation request: The operation does not make a copy of the request. The request can be altered. For example, the PIN_FLD_BILL_OBJ or the PIN_FLD_BILLINFO_OBJ field is inserted according to the PIN_FLD_POID type.
