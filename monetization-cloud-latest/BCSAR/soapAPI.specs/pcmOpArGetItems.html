pcmOpArGetItems

Componentfm_ar
Visibilitypublic
Transactionrequired

Description

This code returns the details of an item object. A transfer event or an item can be specified in the input PIN_FLD_POID parameter. Aggregated amounts will be returned for Billable and Action Items. The following are the ways that details are returned: If an event is given, the operation will return the result of pcmOpArGetItemDetail with no aggregation on the amounts. If a Dispute or Payment Action Item object is specified, the related Settlement or Reversal Item should be provided in the PIN_FLD_RELATED_BILL_ITEM_OBJ field. Item details and currency aggregated amount will be returned. For other Item object, aggregated amounts will be returned for all resource types.
