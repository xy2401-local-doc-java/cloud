This flags is used to truncate or extend the sub balance validity. If PIN_RATE_FLG_CUT is passed then the sub balance validity is truncated to date of cancellation, if PIN_RATE_FLG_MAINTAIN is passed then the sub balance validity period is extended till the account's current billing cycle end date.If PIN_RATE_FLAG_STOP_ROLLOVER is passed, the ROLLOVER_DATA inside the sub_balance bucket will be set to 0.
