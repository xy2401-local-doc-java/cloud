pcmOpArBillSettlement

Componentfm_ar
Visibilitypublic
Transactionrequired

Description

This operation settles a dispute of the bill. One of the following operation operation description messages can be returned to the caller: Item belongs to a different bill. No dispute amount in the item. Item disputed amount and settlement amount must be of same sign. No dispute amount in the bill. Bill disputed amount and settlement amount must be of same sign. The input bill object must be AR bill object.
