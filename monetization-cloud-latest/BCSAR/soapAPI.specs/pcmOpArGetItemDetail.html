pcmOpArGetItemDetail

Componentfm_ar
Visibilitypublic
Transactionrequired

Description

This code returns the details of an item object. A transfer event or an item can be specified in the input PIN_FLD_POID parameter. The following are details are returned: Details of event-level as well as item-level AR action. Action Item as well as Bill Item Transfer details with event comments included for payment reversal and dispute settlement transfer. Tax on Balance Impacts. The PIN_ERRCLASS_SYSTEM_DETERMINATE error buffer returned (if any) can be one of the following: PIN_ERR_BAD_OPCODE PIN_ERR_MISSING_ARG PIN_ERR_BAD_VALUE
