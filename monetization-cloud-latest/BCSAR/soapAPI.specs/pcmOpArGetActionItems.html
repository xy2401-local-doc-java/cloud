pcmOpArGetActionItems

Componentfm_ar
Visibilitypublic
Transactionrequired

Description

This operation returns a list of AR items for a particular bill unit. If an Account is given instead, the operation returns the AR item object information of the default Bill-info object of the Account. Search criteria can be specified in the request.
