pcmOpSubscriptionCancelDiscount

Componentfm_subscription
Visibilitypublic
Transactionrequired

Description

This operation cancels the discount instances associated with the /account object or /service object specified in the request. This operation is also called by pcmOpSubscriptionCancelDeal to cancel each product associated with a specific deal.
