pcmOpArGetAcctActionItems

Componentfm_ar
Visibilitypublic
Transactionrequired

Description

This operation returns a list of A/R items for a particular bill unit. If an Account is given instead, the operation returns the AR items object information of all the bill-info objects of the Account. Search criteria can be specified in the request.

Threshold value for the number of AR Items to be retrieved.
