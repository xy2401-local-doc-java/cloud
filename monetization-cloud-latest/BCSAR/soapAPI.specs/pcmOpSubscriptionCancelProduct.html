pcmOpSubscriptionCancelProduct

Componentfm_subscription
Visibilitypublic
Transactionrequired

Description

This operation cancels the products for the /account object or /service object specified in the request. This operation is also called by pcmOpSubscriptionCancelDeal to cancel each product associated with a specific deal. If the product is part of a required deal, then do not allow the cancellation of it.
