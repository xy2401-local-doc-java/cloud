Array of results information. If writeoff succeed, there'll be one or two elements of the result, depending on whether there's tax to writeoff and PIN_BILL_WRITEOFF_TAX flag is set. Additional event object fields are added to the response here. If the pcmOpflgReadResult flag is set for the operation, all the fields in the event object are returned on the response, not just the poid.
