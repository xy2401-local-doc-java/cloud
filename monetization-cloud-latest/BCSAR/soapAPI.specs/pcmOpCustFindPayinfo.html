pcmOpCustFindPayinfo

Componentfm_cust
Visibilitypublic
Transactionsupports

Description

This operation can be used to locate payinfo objects linked to a particular account and billinfo object. NOTE: If the pcmOpflgReadResult opflag is passed in with this operation, the entire payinfo object is read (any collateral data hanging off of a PIN_FLD_RESULTS array below is ignored).
