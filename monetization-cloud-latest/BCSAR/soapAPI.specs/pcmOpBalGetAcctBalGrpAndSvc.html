pcmOpBalGetAcctBalGrpAndSvc

Componentfm_bal
Visibilitypublic
Transactionrequired

Description

Returns the balance groups and services for all an account's billinfo objects or for a single billinfo object. You can pass flags to get the balance group name and service login aliases.

Flags to determine what to return from a service Valid values: 0: Default. Return balance group and associated services(exclude service-less balance group). 1: Return login/aliases for a service 2: Return the TRANSFER_LIST array 3: Returns default balance group 4: Returns default balance group and service 5: Returns service-less balance group along with other balance groups.
