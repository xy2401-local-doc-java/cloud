pcmOpActActivity

Componentactivity
Visibilitypublic
Transactionrequired

Description

Oracle Monetization Cloud uses pcmOpActActivity to record and rate activity events. This allows Oracle Monetization Cloud to record any type of activity event and any details specific to the event type. Any type of user action can be recorded as an activity event, but it is especially designed to represent events that occur at a single point in time. All activity events are recorded in the system simultaneously.
