pcmOpSubscriptionReadAcctProducts

Componentfm_subscription
Visibilitypublic
Transactionsupports

Description

pcmOpSubscriptionReadAcctProducts input and response specifications. The operation returns the purchased offerings of an account, optionally even the item products and/or cancelled offerings, in a hierarchical service->deal->offering structure.
