pcmOpArGetDisputeDetails

Componentfm_ar
Visibilitypublic
Transactionrequired

Description

This operation returns Dispute Items of a given Bill-info object using the search criteria specified in the request. If a Account is given instead, the operation returns the Dispute Items of the Account's default Bill-info object. Aggregated Amounts per resource will also be returned.
