pcmOpCustDeleteProfile

Componentfm_cust
Visibilitypublic
Transactionrequired

Description

Used to delete the /profile objects. It checks if the /profile object being deleted is used in any /group/sharing/profiles object. If so, then it will not allow the deletion of this /profile object.
