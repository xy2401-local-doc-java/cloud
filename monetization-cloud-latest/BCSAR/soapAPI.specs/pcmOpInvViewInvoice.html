pcmOpInvViewInvoice

Componentfm_inv
Visibilitypublic
Transactionrequired

Description

This operation retrieves an invoice from the database. It uses the poid of the /bill object or /invoice object to locate and retrieve a specific invoice and the PIN_FLD_THRESHOLD value to determine the maximum allowable size of the invoice to be viewed. The PIN_FLD_FLAGS value in the response determines the type of invoice to view, for example a summary or detailed invoice for a non-hierarchical account. The PIN_FLD_INV_SIZE value in the response specifies the size of the invoice returned.
