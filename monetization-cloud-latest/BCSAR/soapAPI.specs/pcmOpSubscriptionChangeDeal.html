pcmOpSubscriptionChangeDeal

Componentfm_subscription
Visibilitypublic
Transactionrequired

Description

This operation wraps pcmOpSubscriptionCancelDeal and pcmOpSubscriptionPurchaseDeal under one transaction giving CSR's or subscribers the ability to change deals in one step. This operation is used in the CSRC Provisioning Interface for giving subscribers the ability to change their service via webkit without cutting themselves off during the process. Although this was designed with CSRC in mind, it can be used for other applications. This operation generates two notification events: /event/notification/deal/change /event/notification/deal/change_complete
