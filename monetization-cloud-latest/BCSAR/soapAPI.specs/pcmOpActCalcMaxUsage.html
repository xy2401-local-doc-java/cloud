pcmOpActCalcMaxUsage

Componentactivity
Visibilitypublic
Transactionrequired

Description

In nutshell, the operation calls pcmOpActUsage with op flag of pcmOpflgCalcOnly, and PIN_FLD_FLAGS of PIN_RATE_FLG_CALC_MAX. It is used to get the maximum quantities possible for authorization. By default, for duration type it limits the maximum duration to 24 hours and for non-duration it limits maximum quantity to a large number 1000000000000000000. To configure the duration maximum, set fm_act max_qty_for_duration entry to a value greater than 24 (hours) in the CM configuration file. Non-duration can not be configured. Refer pcmOpActUsage.xml for other optional input/output entries.
