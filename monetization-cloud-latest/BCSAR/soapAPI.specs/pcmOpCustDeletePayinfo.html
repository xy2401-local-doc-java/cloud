pcmOpCustDeletePayinfo

Componentfm_cust
Visibilitypublic
Transactionsupports

Description

This operation deletes payinfo objects. Note that a payinfo object that is currently associated with a billinfo object cannot be deleted; the billinfo object must be deleted first. For /payinfo/sepa type, the object will not be deleted in-case there is a transaction (/sepa/dd) object in pending status associated with the payinfo.
