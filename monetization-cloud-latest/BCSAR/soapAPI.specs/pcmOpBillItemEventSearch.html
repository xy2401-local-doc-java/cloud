pcmOpBillItemEventSearch

Componentfm_bill
Visibilitypublic
Transactionrequired

Description

pcmOpBillItemEventSearch request specifications: This operation takes an item poid on input and returns the /event objects that belong to the item. This is the default functionality. The operation is also capable of returning sponsored events belonging to the item and transfer events involving the item. Each set of results is returned in a separate result array. The default is to return all fields in the /event object. The caller can optionally restrict this by specifying the fields needed on the results array on input. The caller can additionally restrict the search by time and by event type (a comma delimited list of event type strings can be passed in).
