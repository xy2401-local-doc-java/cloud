pcmOpArResourceAggregation

Componentfm_ar
Visibilitypublic
Transactionrequired

Description

The operation searches event balance impact details and aggregates total per resource. Notes on discount balance impacts: This operation supports Pipeline PIN_IMPACT_TYPE_DISCOUNT and PIN_IMPACT_TYPE_DISCOUNT_RERATED discount balance impacts. The amount and discount will be properly populated. Notes on the operation flags: When the flags include the pcmOpflgLockObj or the pcmOpflgLockDefault flag, they will pass to all pcmOpReadFlds and PIN_OP_SEARCH operations to lock Event objects. The PIN_ERRCLASS_SYSTEM_DETERMINATE error buffer returned can be one of the following: PIN_ERR_BAD_OPCODE PIN_ERR_MISSING_ARG
