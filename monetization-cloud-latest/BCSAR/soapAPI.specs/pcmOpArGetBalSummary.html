pcmOpArGetBalSummary

Componentfm_ar
Visibilitypublic
Transactionsupports

Description

This operation retrieves the unapplied, open bill due, pending bill due, and total dispute balances for default the bill unit (/billinfo object) in a specified account or for a particular bill unit.

0poid    POIDIn Out
Userequired
Class/account
Class/billinfo

Poid of a Bill-info object or an Account object. If an Account object is passed, the operation retrieves the balance summary of the default Bill-info (of the default balance group) of the Account.

0poid    AR_BILLINFO_OBJIn Out
Useoptional
Class/billinfo

Poid of a corresponding AR Bill-info object to the one given in PIN_FLD_POID. If the same poid is specified here, the balance summaries of the Bill-info hierarchy (if any) will be included.

0int    INCLUDE_CHILDRENIn Out
Userequired

Indicator of how the Bill-info hierarchy balance summaries to be returned. Specify a 0 value, if an overall balance summary is desired. Otherwise, detailed balance summaries of the Bill-info hierarchy will be returned.
