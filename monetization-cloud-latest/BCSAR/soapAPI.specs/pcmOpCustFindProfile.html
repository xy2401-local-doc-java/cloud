pcmOpCustFindProfile

Componentfm_cust
Visibilitypublic
Transactionrequired

Description

Used to locate profile objects linked to a particular account object. NOTE: If the pcmOpflgReadResult is passed in with this OP, the entire profile object is read (any collateral data hanging off of PIN_FLD_RESULTS array below is ignored).
