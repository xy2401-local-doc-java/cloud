0enum    MODEIn Out
Useoptional

It indicates the mode of events to be fetched. The values can be 0, 1, 2 and 3. Mode 0 indicates, If event has one or more balance impacts and any one of it matches the criteria, then the event will be fetched. Mode 1 indicates, events will be fetched based on gross charge impact, it excludes balance impacts of type tax (0x4, 0x40) and discount (0x80). Mode 2 indicates, events will be fetched based on net charge impact, it excludes balance impact of type tax(0x4, 0x40) and Mode 3 indicates, event will be fetched based on the total charge impact, it includes charge, tax and discounts. When mode value is greater than zero, the resource id and minimum or maximum value should be passed in the request.
