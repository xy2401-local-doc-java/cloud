pcmOpArGetAcctBills

Componentfm_ar
Visibilitypublic
Transactionrequired

Description

This operation returns the Bill object information of a specified Bill-info. If an Account object is given instead, this operation returns the Bill object information of all the Bill-info of the Account. Search criteria can be specified in the request.
