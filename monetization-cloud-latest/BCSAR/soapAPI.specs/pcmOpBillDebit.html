3int    VALID_FROM_DETAILSIn Out
Useoptional
Parent in.DEBIT_INFO.DEBIT.SUB_BALANCES

The field will have encoded value as lower 8 bit for mode, next 4 bit for unit and remaining 20 bit for offset. The mode can be "First Usage (3)", Absolute (0), Relative (4), Never (2) and Immediate (1). For First Usage Top-up value 3 is passed. Inorder to know when debit was generated, VALID_FROM is used to store with end_t if passed otherwise with the current time.
