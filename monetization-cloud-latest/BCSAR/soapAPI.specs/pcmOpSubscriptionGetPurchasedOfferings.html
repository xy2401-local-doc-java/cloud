pcmOpSubscriptionGetPurchasedOfferings

Componentfm_subscription
Visibilitypublic
Transactionrequired

Description

This operation is used to fetch the purchased products and discounts for a given account, service or billinfo.

Input details

0poid    POIDIn Out
Userequired
Class/account

Account object POID.

0poid    SCOPE_OBJIn Out
Useoptional
Class/account
Class/billinfo
Class/service

Account, Service or Billinfo object that define the scope of the search for products and discounts.

0int    STATUS_FLAGSIn Out
Useoptional

This field specifies the statuses for the offerings. The values of the flags can be: PIN_SUBS_FLG_OFFERING_STATUS_ACTIVE - Fetches only active offerings. PIN_SUBS_FLG_OFFERING_STATUS_INACTIVE - Fetches only inactive offerings. PIN_SUBS_FLG_OFFERING_STATUS_CLOSED - Fetches only closed offerings. Note: Use of multiple values implies the target object should satisfy any of the above.

0timestamp    END_TIn Out
Useoptional

Specifies the date on which the offering is valid. This field is used in conjunction with the validity flags

0int    VALIDITY_FLAGSIn Out
Useoptional

This field specifies the qualifiers for END_T. The values of the flags can be: PIN_SUBS_FLG_OFFERING_VALIDITY_CYCLE - to compare only the CYCLE start and end dates. PIN_SUBS_FLG_OFFERING_VALIDITY_PURCHASE - to compare only the PURCHASE start and end dates. PIN_SUBS_FLG_OFFERING_VALIDITY_USAGE - to compare only the USAGE start and end dates. Note: Use of multiple flags implies the target object should satisfy all of the above.

0int    INCLUSION_FLAGSIn Out
Useoptional

This field specifies additional paramenters to expand the search beyond the scope provided. The values of the flags can be: PIN_SUBS_FLG_INCLUDE_ALL_ELIGIBLE_PRODS - Fetches the products from the given service, its subscription service as well as any account-level products. PIN_SUBS_FLG_INCLUDE_ALL_ELIGIBLE_DISCS - Fetches the discounts from the given service, its subscription service as well as any account-level discounts. PIN_SUBS_FLG_INCLUDE_ALL_SUBSCRIPTION_SERVICES - Includes its subscription services. PIN_SUBS_FLG_INCLUDE_SUBSCRIBER_CONTRACTS - Fetches subscriber contracts for the given scope.

0int    OVERRIDE_FLAGSIn Out
Useoptional

This field specifies additional filters to restrict the search. The values of the flags can be: PIN_SUBS_FLG_ACCT_LEVEL_ONLY - only account level offerings are returned. Valid for Scope = /account only. PIN_SUBS_FLG_OVERRIDE_PRODS_ONLY - This flag is based on value of PIN_FLD_OVERRIDDEN_OBJ. - if the PIN_FLD_OVERRIDDEN_OBJ is NULL, using this flag fetches only the base products. - if the PIN_FLD_OVERRIDDEN_OBJ is a valid poid, using this flag fetches only the overriding products.

0int    SELECT_RESULTIn Out
Useoptional

This field is set to fetch all the records from audit table with the greatest effective_t. The values of the flags can be: 4 - Fetch all records from audit table with the greatest effective_t

0poid    OVERRIDDEN_OBJIn Out
Useoptional
Class/purchased_product

Offering object POID. Only offerings that are overriding this offering will be returned.

0poid    DEAL_OBJIn Out
Useoptional
Class/deal

Deal object POID. Only offerings that are part of this deal are returned.

0int    PACKAGE_IDIn Out
Useoptional

Only offerings for the given package_id are returned. It becomes mandatory if PIN_FLD_INCLUSION_FLAGS contains PIN_SUBS_FLG_INCLUDE_SUBSCRIBER_CONTRACTS.
