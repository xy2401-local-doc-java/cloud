pcmOpCustcareMoveAcct

Componentfm_custcare
Visibilitypublic
Transactionrequired

Description

This is a wrapper operation to move account in or out of an account hierarchy. It was written specifically for Customer Center. The following operations are supported by this operation: 1. Add stand alone account into hierarchy 2. Move account from one hierarchy into other hierarchy 3. Remove the account from hierarchy. It calls these operations: pcmOpBillMakeBillNow, pcmOpBillGroupMoveMember, and pcmOpCustUpdateCustomer when the corresponding array element with the same name is passed.
