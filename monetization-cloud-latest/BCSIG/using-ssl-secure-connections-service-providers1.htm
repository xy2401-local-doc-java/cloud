6 Using SSL to secure connections to service providers

You secure connections between Oracle Monetization Cloud and external service providers, such as payment processors and tax calculators, by using Secure Sockets Layer (SSL) certificates.

Importing SSL certificates

To use SSL to secure connections between Oracle Monetization Cloud and external service providers, import the SSL certificates provided by the service providers.

  1. Open the System Configuration application.
  2. Select SSL Certificate.
  3. In the Upload SSL Certificate screen, select Browse.
    A file upload screen appears.
  4. Select the SSL certificate file provided by the external application, and then select Open.
    The name of the selected file appears next to the Browse button.
  5. In the Alias field, enter a descriptive name for the certificate.
    Don't include spaces in the name.
  6. Select Upload.
    After you upload the certificate, the Oracle Monetization Cloud server automatically restarts to make the certificate effective.
