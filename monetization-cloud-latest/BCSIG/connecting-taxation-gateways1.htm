13 Connecting to taxation gateways

Configuring the Vertex gateway connection

To configure taxation gateway connections, you must have an Oracle Monetization Cloud administrative role.

You configure your Vertex taxation gateway connection by using System Configuration. On the Taxation Gateway page, enter the information described in Vertex gateway information.

Vertex gateway information

Field Description
Base URL The URL for connecting to the taxation web service
Username The user name for your account with the taxation service
Password The password for your account with the taxation service
Cut Off The cutoff value for the taxation service
API Key The unique identifier from the taxation service for API access
