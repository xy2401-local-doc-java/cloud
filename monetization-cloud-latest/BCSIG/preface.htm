Preface

This document explains how to integrate Oracle Monetization Cloud with external business systems by using email messages, event notifications, payment gateways, secure file transfer protocol (SFTP), and web services. It also explains how to test your Oracle Monetization Cloud environment by using virtual time.

Before reading this document, you should be familiar with SOAP API web services and know how to configure RESTful client interfaces to accept REST messages from Oracle Monetization Cloud.

Audience

This document is for developers, system integrators, and other individuals who must integrate external applications with Oracle Monetization Cloud.
