About the REST APIs

Use the Oracle Integration Cloud Service REST API to manage and monitor integrations, adapters, connections, lookups, and packages.

APIs

adapters

Retrieve a list of adapters or details about a single adapter.
View API Endpoints

connections

Retrieve, update, and delete connections.
View API Endpoints

integrations

Retrieve, delete, export, import, activate, deactivate, and run integrations.
View API Endpoints

lookups

Retrieve a list of lookups or details about a single lookup.
View API Endpoints

monitoring

Retrieve integration metrics and error instances and view or export the monitoring log.
View API Endpoints

packages

Retrieve, delete, export, and import packages.
View API Endpoints
