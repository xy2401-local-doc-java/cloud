Creating an Application for Testing an Inbound Adapter

To test an adapter, you must add it to an application. So the first task is to create a new application.

  1. Open JDeveloper.
  2. Select File > New > Gallery.

    Description of the illustration GUID-7B6B1573-AF8B-448F-8F7C-550890D4170D-default.png
  3. Select SOA Application from the gallery.
  4. Type an Application Name.

    Description of the illustration GUID-7C811F85-0421-4654-ACF5-5475D542D5D2-default.png
  5. Type a Project Name.

    Description of the illustration GUID-248A3291-9DDF-40E2-AF0F-F7A9E7BCBCA5-default.png
  6. Click Finish.
An empty composite.xml file is created for the Test project.
See Adding an Exposed Service and Using the Adapter Wizard.
