Adding an External Reference and Using the Adapter Wizard

In JDeveloper, you add an outbound adapter to an application as an external reference.

  1. Drag and drop the SampleCloudAdapter from the Cloud category of the component palette to the External References section.

    Description of the illustration GUID-2A438BC4-6320-4D68-81A7-B35FC2914B1D-default.png
  2. The wizard pops up so you can configure the SampleCloudAdapter. The wizard starts with the Welcome page, which displays the SampleCloudAdapter reference name.

    Description of the illustration GUID-D1A2F4CA-DF11-46DC-A29C-9544A51E5369-default.png
  3. Click Next. The Connection page appears and asks you to provide the target WSDL reference.
    You can also enter security credentials using an Authentication Key.

    Description of the illustration GUID-286D1184-334B-463C-85B3-9EA1C2E48433-default.png
  4. Click Test to test the connectivity to the service.

    A success or failure message is displayed in the Text area.


    Description of the illustration GUID-598BF647-F737-407C-82DD-031118BCC3DD-default.png
  5. Click Next. The Operations Page appears and asks you to select the Operation and its corresponding Business Object.

    Description of the illustration GUID-7BA8561B-4CDF-4E3B-9ECC-90CB05BBF3D9-default.png
  6. Move the desired business object from the Available list to the Selected list.
  7. Click Next. The Summary page appears and displays the items selected during configuration.

    Description of the illustration GUID-02DD289F-36F9-4410-B47A-3F5E08A376CB-default.png
  8. Click Finish to generate the runtime artifacts and references.
    Note: If clicking Finish results in a NullPointerException in the UI thread, it means that you didn’t implement the updateBackEndModel() method for one or more of your wizard pages.

    Description of the illustration GUID-9C778005-9F5D-41D6-9CF6-36CBD7CA425E-default.png

These files are generated:

  • SampleReference.jca (Cloud Application Configuration File)

  • SampleReference.wsdl (Integration WSDL)

See Creating a BPEL Client to Invoke the Adapter.
