Message Transformations for Inbound Adapters

An inbound adapter must often transform messages to and from the source application.

Refer to the SampleMessageReceiver runtime package.

Package, Class, and Methods

Description

Package:oracle.tip.tools.ide.adapters.samplecloudadapter.runtime

Class:

SampleMessageReceiver

This class handles the request and response messages before submitting them to the inbound adapter.

Method:

onMessage()

The SOAP Listener invokes this method to transform the request from an external WSDL to a local WSDL.

run()

Runs the event listener.

activation()

Handles integration flow activation.

release()

Handles integration flow deactivation.

handleResponseMessage()

Handles the response message.

handleRequestMessage()

Handles the request message.
