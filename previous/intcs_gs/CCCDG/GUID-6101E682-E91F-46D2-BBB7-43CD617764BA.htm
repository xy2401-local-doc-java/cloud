Setting Up the REST Cloud Adapter Development Environment

The Sample REST Cloud Adapter project (SampleRESTAdapterProj) has dependencies upon the following JAR files, which are required for successful compilation. You can set up dependency libraries in the project properties to use these JAR files from the ORACLE_HOME location. Or you can extract these JAR files from ORACLE_HOME subdirectories. An example ORACLE_HOME location is C:\Oracle\Middleware\ORACLE_HOME.
