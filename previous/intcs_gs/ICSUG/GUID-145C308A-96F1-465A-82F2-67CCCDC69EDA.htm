What You Can Do from the Salesforce Cloud Invoke Custom Operations Page

You can specify the following values on the Salesforce Cloud invoke Custom Operations page.

  • Select the custom WSDL.
  • Select the operations to perform.
  • Optionally select a new custom WSDL to use.

What You See on the Salesforce Cloud Invoke Custom Operations Page

The following table describes the key information on the Salesforce Cloud invoke Custom Operations page.

Element Description

Select the Custom WSDL

Select the custom WSDL to use.

Operations in Uploaded WSDL

Displays the list of operations included in the uploaded custom WSDL. Select the operation to perform in the Salesforce Cloud application.

Select New Custom WSDL

Click Choose File to select the custom WSDL to use. This selection replaces any previously uploaded WSDL file. After selecting the new WSDL, return to the Operations in Uploaded WSDL list and select the new operation to use.

For more information about using a custom WSDL that includes custom Apex classes written on force.com and exposed as SOAP web services, see Exposing Apex Methods as SOAP Web Services.
