Viewing the Integrations in a Package

You can view the integrations included in a package.

  1. In the Integration Cloud Service toolbar, click Designer.
  2. Click Packages.
    The Packages page is displayed. The package names and the number of integrations included in each package are displayed. You can filter the display of packages by entering the full or partial name (using a wildcard value of *) and clicking the Search icon. If you have not yet created or imported a package into Integration Cloud Service, this page is empty.

    Description of the illustration GUID-69617BF9-2774-4B26-84D8-5DAD4A06B7FD-default.png
  3. Click the name of the package or select View Integrations from the menu at the far right.
    The integrations included in that package and their current states are displayed (for example, pending activation or active).

    Description of the illustration GUID-F98B6A16-38EA-44B0-B9ED-7A7ACECE2AFB-default.png

  4. In the navigator pane, click Integrations to access the Integrations page for viewing these integrations.
