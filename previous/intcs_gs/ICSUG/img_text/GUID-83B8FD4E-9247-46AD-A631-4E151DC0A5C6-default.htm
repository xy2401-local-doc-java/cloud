This image shows a source and target integration. In the source (request) direction, a source adapter connects to a target adapter. Request enrichment mapping and request mapping (which is being selected) icons are configured. In the target (response) direction, the target adapter connects to the source adapter. Response enrichment mapping and response mapping icons are configured. Also in the target (response) direction, a fault mapping icon is available for configuring mapping between the target adapter and source adapter.
