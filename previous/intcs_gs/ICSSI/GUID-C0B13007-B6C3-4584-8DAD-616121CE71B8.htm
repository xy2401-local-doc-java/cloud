Configuring Oracle Siebel Adapter Trigger Request Properties

Enter the Oracle Siebel Adapter trigger request parameters.

Topics

  • What You Can Do from the Oracle Siebel Adapter Trigger Request Properties Page

What You Can Do from the Oracle Siebel Adapter Trigger Request Properties Page

You can configure the following parameters on the Oracle Siebel Adapter trigger request properties page.

  • Select from the web services that are published.
  • Select an operation type for the web service that you have selected.
  • View the description of the web service that you have selected.

What You See on the Oracle Siebel Adapter Trigger Request Properties Page

The following table describes the key information on the Oracle Siebel Adapter trigger Request properties page.

Element Description

Select a Published Service

Siebel published web service to invoke.

Select an Operation

Select an operation from the published web service.

Web Service Description

Displays the description of the web service that you have selected.

Changing Service Transport And Binding of Web Services in Siebel Application

If you are using Siebel Adapter on ICS inbound, you might not be able to generate WSDL for a few web services. This happens because of service configuration issues (Transport and Binding) on the Siebel Application.

If you are using Siebel Adapter on ICS inbound, you might not be able to generate WSDL for any of the following services:

  • External Account

  • External Contact

  • External Employee

  • External Order

  • External Position

  • External Quote

  • External ATP Check

  • External Credit Check

  • External Shipment Tracking

  • External Simple Product

This happens because of service configuration issues (Transport and Binding) on the Siebel Application. To resolve, change Transport and Binding of Siebel service. To change Transport and Binding information of Siebel Service:

  1. Login to Siebel call center.
    For example, http://<SiebelHost>/callcenter_enu.
  2. Click on Navigate, Site Map.
    Screens page with all the available screens is displayed.
  3. Click Administration – Web Services.
    Web Services page is displayed.
  4. Click Outbound Web Services under Administration – Web Services.
    Outbound Web Services screen is displayed.
    Description of the illustration GUID-9C54DD35-1877-4EB4-B98B-0FC9D730C8D1-default.png
  5. Select the services for which you need to change the Transport and Binding and make the following changes for each service:
    1. Click Transport and from the list options, and select HTTP.
    2. Click Binding and from the list options and select SOPA_DOC_LITERAL.
