What You Can Do from the Evernote Adapter Operation Selection Page

You can configure the following parameter on the invoke Evernote Adapter Operation Selection page.

  • Select the Evernote API operation to perform.

What You See on the Evernote Adapter Operation Selection Page

The following table describes the key information on the Evernote Adapter Operation Selection page.

Element Description

Select Operation

Select the Evernote API operation to perform. The operations that are displayed are based on the API permission access level that you selected on the Evernote Developers site as, described in Prerequisites for Creating a Connection.
