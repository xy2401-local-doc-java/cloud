Building Mapping Statements

This section describes how to build mapping statements.

To build mapping statements:
  1. Drag a source to a target to create a mapping.
  2. Click the target node to launch the Mapping Builder.
    If the target is not mapped, you see two statements: the target tag and a new empty statement with the text Drag and Drop or Type here….
  3. Drag a source element onto the empty row.

    Description of the illustration GUID-08CC92B5-1F6D-48A3-B10D-4CBAE8FAD3C6-default.png
    The value-of mapping statement appears.

    Description of the illustration GUID-36CC794E-6562-46DD-A35F-E3DFC1D662F4-default.png

  4. Click the Edit icon, or right click and select Edit.
    The statement expands to show the select and source statements.
  5. Change the mapping as necessary by dragging a different source element onto the source statement.
    Description of the illustration GUID-485E3A5B-63EA-4C91-8752-D56B25285D9A-default.png
  6. Click Save.
