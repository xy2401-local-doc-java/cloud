Building Expressions with the Mapping Builder

Use the Mapping Builder to create and modify advanced mappings. For example, you can loop through repeating elements, create conditional statements, use XPath functions, and reference lookups. The Mapping Builder is displayed when you click a field in the target data structure of the mapper.
