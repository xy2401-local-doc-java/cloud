4 Adding the Oracle RightNow Cloud Adapter Connection to an Integration

When you drag the Oracle RightNow Cloud Adapter into the trigger and invoke areas of an integration, the Adapter Endpoint Configuration Wizard appears. This wizard guides you through configuration of Oracle RightNow Cloud Adapter endpoint properties.

The following sections describe the wizard pages that guide you through configuration of the Oracle RightNow Cloud Adapter as a trigger and invoke in an integration.
