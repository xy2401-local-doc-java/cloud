About the Microsoft Contact Adapter

Use the Microsoft Contact Adapter to transfer files to a target Microsoft Contact server in an Oracle Integration Cloud Service integration.

The Microsoft Contact Adapter is one of many predefined adapters included with Oracle Integration Cloud Service. You can configure the Microsoft Contact Adapter as a connection in an integration in Oracle Integration Cloud Service. For information about Oracle Integration Cloud Service, connections, and integrations, see the following sections:
