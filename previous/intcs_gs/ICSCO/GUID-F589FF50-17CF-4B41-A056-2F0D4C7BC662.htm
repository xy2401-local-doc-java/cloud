What You Can Do from the Concur Adapter Invoke Operations Page

You identify the operation type and the operations to perform in the Concur application.

What You See on the Concur Adapter Invoke Operations Page

The following table describes the key information on the Concur Adapter Operations page.

Element Description

Select Operation Type

Selects the API operation type. These are the available operations:

  • Create

  • Update

  • Get

  • Delete

Select Operation

Selects the API operation to perform.

Description

A description of the API operation that is performed.
