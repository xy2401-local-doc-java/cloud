About the DB2 Adapter

The DB2 Adapter enables you to run stored procedures or SQL statements against an IBM DB2 database as part of an integration in Oracle Integration Cloud Service.

The DB2 Adapter provides the following benefits:

  • You can run stored procedures.

  • You can run SQL statements.

  • Distributed polling and multithreading support is provided.

The DB2 Adapter is one of many predefined adapters included with Oracle Integration Cloud Service. You can configure the DB2 Adapter as a connection in an integration in Oracle Integration Cloud Service.
