4 Adding the Oracle ERP Cloud Adapter Connection to an Integration

When you drag the Oracle ERP Cloud Adapter into the trigger and invoke areas of an integration, the Cloud Endpoint Configuration Wizard is invoked. This wizard guides you through configuration of the Oracle ERP Cloud Adapter endpoint properties.

The following sections describe the wizard pages that guide you through configuration of the Oracle ERP Cloud Adapter as a trigger or invoke in an integration.
