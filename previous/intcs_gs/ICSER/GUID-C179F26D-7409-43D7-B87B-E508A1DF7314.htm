Enabling Oracle ERP Cloud Events in Integrations

You must perform the following tasks to enable the display of Oracle ERP Cloud events in integrations. The Product inventory module (PIM) exposes public APIs and events in release 11 of Oracle ERP Cloud.

To successfully subscribe to events, a CSF key is required. This key must be created on the Oracle ERP Cloud side. Create the CSF key in either of two ways:

  • File a service request with Oracle Support Services to create the CSF key in the Oracle Fusion Applications Supply Chain Management (SCM) domain. This action enables product inventory module (PIM)-published events to be visible for selection in Oracle Integration Cloud Service. In Oracle ERP Cloud 11, PIM events are the only events supported. Your Oracle Integration Cloud Service identity domain must be used as the name of the CSF key.

  • If you are using release 12, you can manually create the CSF key in Oracle SOA Composer. Oracle SOA Composer is accessible from the Supply Chain Management Cluster.

    1. Manually add a special permission to the SOA server domain's system-jazn-data.xml file:

      <grant>
   <grantee>
      <codesource>
            <url>file:${domain.home}/servers/${weblogic.Name}/tmp/_WL_user/soa-infra/-</url>      
      </codesource>
   </grantee>
   <permissions>
       <permission>
            <classoracle.security.jps.service.credstore.CredentialAccessPermission</class>
            <name>context=SYSTEM,mapName=oracle.wsm.security,keyName=*</name>
            <actions>*</actions>
       </permission>
   </permissions>
</grant>

      Without these permissions, you receive the following error and must file a service request with Oracle Support Services to create the CSF key.

      "access denied (oracle.security.jps.service.credstore.CredentialAccessPermission context=SYSTEM,mapName=oracle.wsm.security,keyName=* read)"

    2. Restart the server.

    3. Log in to Oracle SOA Composer with a user that has the SOA administrator role. Obtain the hostname and port from your administrator.

      http://CRM_domain_URL:port/soa/composer

    4. Click Manage Security.

    5. Add the csf-key.

    6. Provide the username and password for Oracle ERP Cloud to use to invoke Oracle Integration Cloud Service subscriptions.
