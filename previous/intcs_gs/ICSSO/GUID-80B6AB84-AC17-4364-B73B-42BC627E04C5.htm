4 Adding the SOAP Adapter Connection to an Integration

When you drag the SOAP Adapter into the trigger and invoke areas of an integration, the Cloud Endpoint Configuration Wizard is invoked. This wizard guides you through configuration of the SOAP Adapter endpoint properties.

The following sections describe the wizard pages that guide you through configuration of the SOAP Adapter as a trigger or invoke in an integration.

Topics

  • Configuring Basic Information Properties

  • Configuring SOAP Adapter Trigger Operation Properties

  • Configuring SOAP Adapter Invoke Operation Properties

  • Configuring SOAP Adapter Request Header Properties

  • Configuring SOAP Adapter Response Header Properties

  • Reviewing Configuration Values on the Summary Page

For more information about the SOAP Adapter, see SOAP Adapter Capabilities.
