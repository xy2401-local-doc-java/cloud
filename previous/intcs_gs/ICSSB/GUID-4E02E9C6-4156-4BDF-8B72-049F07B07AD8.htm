About the Oracle Database Adapter

The Oracle Database Adapter enables you to run stored procedures or SQL statements on a database as part of an integration in Oracle Integration Cloud Service.

The Oracle Database Adapter provides the following benefits:

  • You can run stored procedures. Support is provided for non-JDBC (PL/SQL) datatypes in outbound invocations of stored procedures.

  • You can run SQL statements.

  • Distributed polling and multithreading support.

The Oracle Database Adapter is certified to connect to Oracle Database Cloud Service through the on-premises agent. You install the on-premises agent on the Oracle Database Cloud Service instance. On the Connections page, you then provide Oracle Database Cloud Service instance-specific connection parameters, including the host, port, service ID, username, and password. You associate the respective agent group and then test the connection. For information about installing the on-premises agent, see Downloading and Running the On-Premises Agent Installer of Using Oracle Integration Cloud Service. For information about Oracle Database Cloud Service, see Oracle Database Cloud - Database as a Service.

The Oracle Database Adapter is one of many predefined adapters included with Oracle Integration Cloud Service. You can configure the Oracle Database Adapter as a connection in an integration in Oracle Integration Cloud Service.
