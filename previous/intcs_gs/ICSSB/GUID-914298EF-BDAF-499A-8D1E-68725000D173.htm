4 Adding the Oracle Database Adapter Connection to an Integration

When you drag the Oracle Database Adapter into an integration, the Adapter Endpoint Configuration Wizard appears. This wizard guides you through configuration of Oracle Database Adapter endpoint properties.

The following sections describe the wizard pages that guide you through configuration of the Oracle Database Adapter as a trigger and an invoke in an integration.

Topics

  • Configuring Basic Information Properties

  • Configuring Oracle Database Adapter Invoke Stored Procedure Properties

  • Configuring Oracle Database Adapter Invoke SQL Statement Properties

  • Configuring Oracle Database Adapter Trigger Polling Properties

  • Reviewing Configuration Values on the Summary Page

For general information about the Oracle Database Adapter, see About the Oracle Database Adapter.

Note:

  • The Oracle Database Adapter does not support the regeneration of WSDL artifacts. For more information about regeneration, see Regenerating a WSDL File for Integrations (in Using Oracle Integration Cloud Service).

  • You cannot edit the existing trigger Oracle Database Adapter settings or delete or re-import the previously configured Oracle Database Adapter tables. To modify an existing integration, delete the connection, drag the Oracle Database Adapter to the trigger or invoke area of the integration, and reconfigure the Oracle Database Adapter endpoint properties.
