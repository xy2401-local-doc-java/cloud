4 Adding the Oracle Responsys Adapter Connection to an Integration

When you drag the Oracle Responsys Adapter into the target area of an integration, the Adapter Configuration Wizard appears. This wizard guides you through configuration of Oracle Responsys Adapter endpoint properties.

The following sections describe the wizard pages that guide you through configuration of the Oracle Responsys Adapter as a target in an integration. The Oracle Responsys Adapter cannot be used as a source in an integration.
