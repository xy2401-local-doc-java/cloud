4 Adding the Microsoft Calendar Adapter Connection to an Integration

When you drag the Microsoft Calendar Adapter into the invoke area of an integration, the Adapter Endpoint Configuration Wizard is invoked. This wizard guides you through configuration of Microsoft Calendar Adapter endpoint properties.

The following sections describe the wizard pages that guide you through configuration of the Microsoft Calendar Adapter as an invoke in an integration. The Microsoft Calendar Adapter cannot be used as a trigger in an integration.

Topics
