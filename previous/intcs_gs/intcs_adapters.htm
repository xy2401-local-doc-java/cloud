Oracle Integration Cloud Service

Adapters

Oracle provides a number of adapters for use with Oracle Integrated Cloud Service. The following list shows the different adapter categories.  Click the link under a category to see a list of adapters and read about how they work.



Adapter Development

  • Create custom adapters using the Cloud Adapter SDK.

CRM/CX Application Adapters

  • Manage customer data and interaction; access business data; automate sales, marketing, and customer support; and manage employee, vendor, and partner relationships.

ERP Application Adapters

  • Collect, store, manage, and analyze data from many business activities, including product planning, purchasing, manufacturing, or service delivery.

Financial Adapters

  • Work smarter, increase efficiency and reduce back-office costs with standardized processes for shared services, productivity tools, and integrated performance management.

Supply Chain Management Adapters

  • Innovate quickly, execute rapidly, and scale your value chain. Companies can rapidly deploy Oracle's supply chain management functionality with minimal risk, low cost, and maximum flexibility

HCM Adapters

  • Define and communicate employee performance expectations. Rate, reward, and hold employees accountable for achieving goals, creating innovation, and supporting continuous improvement.

On-Premises Application Adapters

  • Use with the agent framework to create integrations and exchange messages between on-premises applications and Oracle Integration Cloud Service.

SaaS Application Adapters

  • Create software as a Service (SaaS) application integrations.

Social/Productivity Adapters

  • Create, update, share, or exchange information with various social networking sites.

Technology Adapters

  • Interact with file systems, FTP servers, Java Message Services (JMS), database tables, REST resources, web services, and other technologies.

Industries Adapters

  • Target the needs of users with skill sets and job responsibilities aligned to a specific business industry or market.
