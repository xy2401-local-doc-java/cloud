Deploying MyCloudApplication

The MyCloudAdapter is an adapter for MyCloudApplication, so before you can use MyCloudAdapter, MyCloudApplication must be deployed.

  1. Go to the WebLogic Administration Console:
    http://hostname:port/console
  2. Under Domain Structure, select Deployments.

    Description of the illustration GUID-C74D614D-2DEE-4ED6-9224-0C0EF51EC8EF-default.png
  3. On the Summary of Deployments page, click Install.

    Description of the illustration GUID-3D9FDE2B-C25C-420D-AD1B-9EA356F21253-default.png
  4. In the Install Application Assistant, click upload your file(s).

    Description of the illustration GUID-55560520-B49E-482D-9048-3E89BE4A4D3A-default.png
  5. For Deployment Archive, click Browse and select MyCloudApplication.war.

    Description of the illustration GUID-1FF45345-A48F-4561-85F8-814549103FC1-default.png
  6. Click Next.
  7. Make sure the WAR file is uploaded successfully.
  8. Click Next.
  9. Under Choose Targeting Style, click Install this deployment as an application.

    Description of the illustration GUID-ECCAA079-3E53-468F-86FF-552DDD742122-default.png
  10. Click Next.
  11. Use default values for the optional settings.
  12. Click Finish.

    Description of the illustration GUID-EE1E8E55-C822-4BB5-AD95-2009C7DC51AC-default.png

After deployment, the WSDL file is located here:

http://hostname:port/MyCloudApplication/MyCloudAppService?WSDL
Next, see Registering MyCloudAdapter with Jdeveloper.
