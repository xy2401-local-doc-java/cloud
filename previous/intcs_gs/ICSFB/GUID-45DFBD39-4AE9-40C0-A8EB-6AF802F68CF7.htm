4 Adding the Facebook Adapter Connection to an Integration

When you drag the Facebook Adapter into the invoke area of an integration, the Adapter Endpoint Configuration Wizard appears. This wizard guides you through configuration of Facebook Adapter endpoint properties.

The following sections describe the wizard pages that guide you through configuration of the Facebook Adapter as an invoke in an integration. The Facebook Adapter cannot be used as a trigger in an integration.
