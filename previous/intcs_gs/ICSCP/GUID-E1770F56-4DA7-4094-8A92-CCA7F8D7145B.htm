What Application Version Does the Oracle CPQ Cloud Adapter Support?

The Oracle CPQ Cloud Adapter was originally created to work with 2015R1, but works with any later version of Oracle CPQ Cloud. The only tightly coupled case is with Oracle Commerce Cloud integration, where 2016R2 for Oracle CPQ Cloud is required along with whichever version of the Oracle CPQ Cloud Adapter is released with it.
