4 Adding the Oracle Commerce Cloud Adapter Connection to an Integration

The Oracle Commerce Cloud Adapter enables you to create integrations with Oracle Commerce Cloud web hooks and endpoints. When you drag the Oracle Commerce Cloud Adapter into the trigger or invoke area of an integration, the Adapter Endpoint Configuration Wizard appears. This wizard guides you through configuration of Oracle Commerce Cloud Adapter endpoint properties.

The following sections describe the wizard pages that guide you through configuration of the Oracle Commerce Cloud Adapter as a trigger or invoke in an integration.

Topics
