4 Adding the File Adapter Connection to an Integration

When you drag the File Adapter into the trigger or invoke area of an integration, the Adapter Endpoint Configuration Wizard appears. Use the wizard to configure File Adapter endpoint properties.

These topics describe the Adapter Configuration Wizard pages that you use to configure the File Adapter as a trigger and an invoke in an integration.
