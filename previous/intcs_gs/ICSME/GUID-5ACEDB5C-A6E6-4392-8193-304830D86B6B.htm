About Microsoft Email Adapter Use Cases

The Microsoft Email Adapter can be used in the following scenarios.

  • You can integrate a Microsoft email account with the Microsoft Email Adapter to return information such as emails, attachments, and folders

  • You can integrate a Microsoft email account with the Microsoft Email Adapter to complete tasks such as sending email, creating new folders, deleting email and attachments, and creating new file attachments.
