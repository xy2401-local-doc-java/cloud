4 Adding the Microsoft Email Adapter Connection to an Integration

When you drag the Microsoft Email Adapter into the invoke area of an integration, the Adapter Endpoint Configuration Wizard is invoked. This wizard guides you thorough configuration of Microsoft Email Adapter endpoint properties.

The following sections describe the wizard pages that guide you through configuration of the Microsoft Email Adapter as an invoke in an integration. The Microsoft Email Adapter cannot be used as a trigger in an integration.
