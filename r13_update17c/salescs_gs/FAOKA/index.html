About the REST APIs

REST APIs for Knowledge in Oracle Engagement Cloud allow you to access knowledge data in Engagement Cloud and provide a way to integrate other systems. You can use these REST APIs to create and edit knowledge articles; search for knowledge articles; make recommendations; and view user information from external applications. These REST APIs adhere to Oracle REST standards.
