Oracle CASB Cloud Service

Get Started

Oracle CASB (Cloud Access Security Broker) Cloud Service enables enterprises to realize the full promise of the cloud, both for consuming services from and delivering services through the cloud. Oracle CASB Cloud Service includes support for IaaS including Amazon Web Services (AWS) and Rackspace, as well as support for SaaS applications including Box, GitHub, Google Apps, Microsoft Office 365, ServiceNow, and Salesforce.

Product names used in this documentation are: Oracle CASB Cloud Service (previously Palerra LORIC) and CASB for Discovery (previously Palerra LORIC Discovery).
