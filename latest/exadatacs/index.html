Exadata Cloud Service

Get Started

Oracle Database Exadata Cloud Service provides full Oracle Databases hosted on Oracle Exadata Database Machine. Exadata Cloud Service is offered on Oracle Cloud, using state-of-the-art Oracle-managed data centers. You can also choose Exadata Cloud at Customer, which provides Exadata Cloud Service hosted in your data center.

Use the following topics to get started, or use the menu to explore everything from what's new to other documentation.

Oracle Database Exadata Cloud Service is just one of the service offerings that provide Oracle Database in the cloud. Visit cloud.oracle.com/database to learn more about all of the Oracle Database Cloud services.
