How do I show or hide membership messages in a collection?

When a user is added to or removed from a collection a membership message is posted to the collection (for example, The collection is now accessible to: John Doe). If you'd prefer not to see those messages, you can hide them.

Click , then select Hide Membership Messages or Show Membership Messages. For more information, see How do I show or hide membership messages?
