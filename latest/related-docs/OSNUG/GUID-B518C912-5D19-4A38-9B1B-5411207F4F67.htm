How do I mark a message as read or unread?

Within a Conversation, unread messages are marked to the left of the message. Click the blue dot to mark a single message as read.

To mark all the messages in a Conversation as read, click the blue icon that contains a number, like this: . You'll find this unread message count in many places throughout Oracle Social Network.

To mark a message as unread, click More under the message, then click Mark as Unread.
