How do I add a related item to a Social Object?

You can add an existing item or create a new one.

  1. Open the Social Object.
  2. Tap in the action bar to display the view menu, then tap Related.
  3. Tap to open the Add Related dialog.

    Tip: Tap to choose between Recent, Favorites, or Search to find the collection or Social Object.

  4. Select an existing item or create a new one:

    • To select an existing item, tap the item you want to relate to the open Social Object.

    • To add a new Conversation, tap in the action bar, enter a new name for the Conversation, then tap OK.

    • To add a new collection, tap in the action bar, enter a new name for the collection, then tap OK.
