How do I add a referring item to a Conversation?

This action creates a relationship between the Conversation and the item you select, making it easy to navigate between the two. You can add an existing item or add a new one.

  1. Open the Conversation.
  2. Tap in the action bar to display the view menu, then tap Referring.
  3. Tap to open the Add Related dialog.

    Tip: Tap to choose between Recent, Favorites, or Search to find the collection or Social Object.

  4. Tap the collection or Social Object to add.

    Alternatively, you can create a new collection. Tap in the action bar, enter a new name for the collection, then tap OK.
