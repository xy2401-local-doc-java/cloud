How can I search for a term within a single Conversation?

The Outlook add-in provides controls for this:

  1. Open the Conversation you want to search.

  2. Click Find messages in Conversation, which is the icon that looks like a magnifying glass.

  3. In the Find drop-down, select Messages.

  4. Enter the search term in the Text field.

  5. Use the up and down arrows to step through the search results.

If you're using Oracle Social Network in your browser, it's still possible to search for individual keywords across all Conversations:

  1. Enter the search term in the search field at the top.
  2. Press Enter or click the search icon to display the full-text search results.
  3. In the search results, click Conversations on the left.
  4. Find the Conversation you want, then click Show more messages to see all the messages containing the keyword.
