Oracle API Manager Cloud Service

Get Started

NOTE: API Manager Cloud Service is no longer available to new customers. Instead, please investigate API Platform Cloud Service.

Oracle API Manager Cloud Service facilitates the creation of APIs that expose the functionality of backend systems or other services. These APIs are published for use by application developers and are managed and monitored at runtime. You provision Oracle API Manager Cloud Service as part of the Oracle SOA Cloud Service provisioning wizard.

Note that Oracle API Manager must be licensed separately from Oracle SOA Cloud Service, but is provisioned as a service type under the Oracle SOA Cloud Service umbrella.  As such, you obtain both trial subscriptions and regular subscriptions through SOA Cloud Service.  You choose "API Manager Cluster" during the provisioning process.
