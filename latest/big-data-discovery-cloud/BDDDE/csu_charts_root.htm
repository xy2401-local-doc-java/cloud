24 Charts

Studio provides chart components that you can use to visualize data and the distribution of data values. The chart components include the Chart, Box Plot, Histogram Plot, and the Parallel Coordinates Plot.

  • Chart
    The Chart component displays a graphical chart based on the project data. It supports several sub-types and includes options for selecting the specific data to display.
  • Histogram Plot
    A Histogram Plot is a basic visualization for showing the distribution of values for a single metric.
  • Box Plot
    A Box Plot provides a capsule summary of metric values for each value of a selected String attribute.
  • Parallel Coordinates Plot
    A Parallel Coordinates Plot displays the values of several metrics for each value of a single grouping dimension.
