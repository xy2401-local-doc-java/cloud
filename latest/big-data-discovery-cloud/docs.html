Cloud

Oracle Big Data Discovery Cloud Service

Books

Documentation for Oracle Big Data Discovery Cloud Service (BDDCS). Learn how to find, explore, discover, transform, and share your big data in the cloud, including how to create, start, and stop a BDDCS instance, back it up, apply a patch and restore from backup. And, you'll see how to use Studio to find, explore, visualize, and transform Hive data, as well as how to load new or update existing data.
