Creating an Implementation Plan

During Functional Configuration, Implementors use Functional Setup Manager to enable Offerings and create an Implementation Plan.

To perform Functional Configuration:
  1. Enable your Offerings:
    1. In Configuration Manager, select the Perform Functional Configurations link on the Tasks pane.
      Description of GUID-D9E6463C-EB28-4D5E-9557-601AE840392C-default.gif
      If prompted, enter the user name and password that you used to log into Configuration Manager. Functional Setup Manager is displayed.

      Description of GUID-23B0EA9B-DF06-4B69-B068-B0582ACF657E-default.gif
    2. In Functional Setup Manager, select the Configure Offerings link to display the Configure Offerings dialog.
    3. Select the Enable for Implementation check box next to the Offering that you are deploying, for example, OTBI Enterprise for HCM Cloud Service.

      Description of GUID-EAD1C079-1E78-4F54-8204-3DBE405D72D6-default.gif
    4. Click Save and Close.
  2. Create an Implementation Plan:
    1. In Functional Setup Manager, select the Manage Implementation Projects link on the Tasks pane.
    2. Click the Create icon (or select Actions, then Create), use the Enter Basic Information dialog to specify a Name for the Implementation Plan, then click Next.
    3. At the Select Offerings to Implement page, select the Include check box next to the Offering that you are deploying, for example, OTBI Enterprise for HCM Cloud Service.

      Description of GUID-C23EB661-340A-4A5C-83B8-FCAB56D867CD-default.gif
    4. Click Save and Open Project.

      Description of GUID-0ECD8C33-64DF-46EB-A396-EE7A975FCBF1-default.gif
You can now proceed to the next step where you create a Configuration Workbook.
