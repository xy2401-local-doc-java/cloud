6 Monitoring and Diagnosing Data Load Issues

During the Loading Data stage, Implementors use Configuration Manager to monitor the progress of load plans. If a data load fails or completes with auto-corrections, then the Implementor receives an email alert that contain a summary of the data loading issue.
