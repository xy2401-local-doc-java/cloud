Enabling Offerings for Deployment

During Initial Configuration, OTBI-Enterprise Implementors specify the Cloud data source details. Before you start, log in to Configuration Manager using the Web link and user name supplied to you by Oracle Cloud Support.

  1. In Configuration Manager, select the Manage Business Intelligence Applications link in the System Setups area on the Tasks pane.

  2. Select the Enabled option next to the offering and functional areas that you are deploying.
  3. Click Save, then Done.
