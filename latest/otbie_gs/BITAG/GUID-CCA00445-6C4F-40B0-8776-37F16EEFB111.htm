About Administration Tools

Administrators and implementors with the correct privileges can access administration tools from links on the left side of the Oracle BI Answers user interface.

Administration tools include:

  • Configuration Manager: Used by implementors to perform setup and configuration, maintenance and functional setups, and to create, manage, schedule and execute load plans to load data into the OTBI Enterprise data warehouse for analysis by BI end users.

  • Service Console: Used by administrators to manage users and roles, perform administrative tasks, and to monitor and view metrics about a service.

  • ODI Console: Used by implementors and administrators to view details about load plan execution and to monitor data loads to the OTBI Enterprise data warehouse.

For information about the administrative roles and associated tool access privileges, see About Required Enterprise Roles.
