Oracle Java Cloud Service - SaaS Extension

Get Started

Oracle Java Cloud Service - SaaS Extension reduces the complexity associated with deploying and maintaining enterprise Java applications. With it, you can create Oracle Java Cloud Service - SaaS Extension instances quickly, easily deploy your applications to them, and then secure and manage the applications without worrying about the underlying infrastructure. Powered by Oracle WebLogic Server, Oracle Java Cloud Service - SaaS Extension is a part of the platform service offerings in Oracle Cloud.
