                                                                                                              Overview

                                                                                                              About Oracle Push Cloud Service (formerly Push IO):

                                                                                                              This is the document for mobile app developers using Oracle Push Cloud Service - Standalone.

                                                                                                              Oracle Push Cloud Service is a SaaS push notifications offering, and has delivered billions of push notifications and currently powers some of the most widely used apps.

                                                                                                              Our services go beyond sending simple push messages. The enterprise class platform offers robust querying and analytics tools, helping the world's best marketers deliver relevant notifications that stimulate engagement, and drive users back to your applications.
