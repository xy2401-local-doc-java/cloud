    About Oracle Responsys Mobile App Platform Cloud Service (formerly Oracle Push Cloud Service - Integrated with Responsys)

    This is the document for mobile app developers using Oracle Responsys Mobile App Platform Cloud Service.

    Oracle Responsys Mobile App Platform Cloud Service is a SaaS push notifications offering, and has delivered billions of push notifications and currently powers some of the most widely used apps.

    Our services go beyond sending simple push messages. This enterprise-class platform helps modern marketers deliver relevant push and in-app notifications that stimulate engagement, and drive users back to your applications.
