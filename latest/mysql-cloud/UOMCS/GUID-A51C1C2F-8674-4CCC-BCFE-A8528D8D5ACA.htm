Creating and Managing IP Reservations

This topic does not apply to Oracle Cloud Infrastructure.

If your identity domain supports the regional compute infrastructure, you can create and manage IP reservations. You use these reservations when creating MySQL Cloud Service deployments instead of having MySQL Cloud Service assign one for you.
