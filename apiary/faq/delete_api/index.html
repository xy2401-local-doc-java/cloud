How can I delete an API in Apiary?

  1. Go to the settings page for the API:

  2. Scroll to the section “API Domain” at the top and remember or copy the subdomain.

  3. Go to the section “Remove API Description”

  4. Fill in or paste the API Domain name into the input field you found under the API Domains section

  5. Click the button “I know what I am doing. Remove this API”:

Note
  • this operation can’t be undone
  • you can’t delete the last API under your account
